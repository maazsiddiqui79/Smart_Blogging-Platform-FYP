Privacy Policy

Effective Date: 01 Jan 2026

This Privacy Policy explains how Smart Blogging collects, uses, stores, and protects user information. By using this platform, you agree to the practices described in this policy.

Smart Blogging is an academic final year project developed for educational purposes. While reasonable security measures are implemented, the platform is not a commercial service.

1. Information We Collect

When you create an account or use the platform, we may collect:

  • Full name and username
  • Email address
  • Profile image and biography
  • Published blog posts and comments
  • Follower and like activity
  • Login session information
  • Basic device or IP address information for security purposes

2. How We Use Your Information

Your information is used to:

  • Create and manage user accounts
  • Display public profiles and blog content
  • Improve user experience
  • Maintain platform security
  • Analyze usage for academic evaluation

3. Publicly Visible Information

Certain information is visible to other users and the public, including:

  • Username
  • Profile image
  • Bio
  • Published blogs
  • Follower and engagement counts

Users are advised not to publish sensitive personal data.

4. Data Security

Smart Blogging uses standard security practices to protect user data.

  • Passwords are securely hashed using Django’s authentication system.
  • Administrative access is restricted.
  • Reasonable safeguards are applied to prevent unauthorized access.

However, no internet-based system can guarantee absolute security.

5. Cookies and Session Data

The platform uses cookies and session technologies to:

  • Maintain login sessions
  • Remember user preferences
  • Improve overall functionality

Users may control cookie usage through browser settings.

6. Data Sharing

Smart Blogging does not sell, rent, or trade user data.

Information may only be disclosed:

  • If required by law
  • For academic review or project evaluation purposes

7. Third-Party Services

The platform may include links to third-party websites. Smart Blogging is not responsible for the privacy practices or content of external services.

8. User Rights

Users have the right to:

  • Update profile information
  • Edit or delete blog posts
  • Request account deletion
  • Request removal of personal data

9. Data Retention

User data is retained for as long as the account remains active or as required for academic evaluation purposes. Upon account deletion, reasonable steps will be taken to remove associated data.

10. Updates to This Policy

This Privacy Policy may be updated periodically. Continued use of Smart Blogging after updates indicates acceptance of the revised policy.
